# MLX Vision-Language Model Benchmark

Decode throughput and memory usage across context lengths with and without TurboQuant KV cache quantization.

**Hardware:** Apple M5 Max (128 GB unified memory)
**Framework:** mlx-vlm 0.4.4, MLX 0.31.1
**Context:** 4k to 256k tokens | Thinking: OFF
**TurboQuant:** KV cache quantized to 2.5 bits

See the [interactive HTML report](results/benchmark_report.html) for charts.
